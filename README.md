Accompnaying homework assignments for the lecture series done at Shuttl by me.

# Setup
You will need to have haskell setup for you to work on it. There are many ways to go about doing it I will share a beginner friendly way to set this up

## Setup Haskell on your machine
1. Get Stack `curl -sSL https://get.haskellstack.org/ | sh`
2. `stack ghci`
3. Try out `1 + 1` in the repl
Note: https://docs.haskellstack.org/en/stable/README/#how-to-install

## Setup this project
1. fork this repo
2. clone this repo
3. cd `learning-haskell`
4. `stack test`
5. Say yay (optional)
